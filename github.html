TRY IT BY YOURSELF


    You are working in Visual Studio? -> Make few steps :


  1. Open Terminal in Visual Studio Code.
  2. You need to put the command cd to create a dir.
  3. Open your account in Github.
  4. You need to create new repository for example manually.
  5. Make o copy of the link in your repository.
  6. Come back to the Visual Studio Code - Terminal.
  7. Make sure you are connected with your Github's account - write command git init.
  8. Next command is git add .
  9. Is good to check the status also writing git status.
  10. Write git commit -m "description of your shipment".
  11. Last command is git push origin master.
  12. If you did this corectly, you should see the updated repositoryin your account.
